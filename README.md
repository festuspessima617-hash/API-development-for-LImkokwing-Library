#  Library Management System

## Description

This is a simple Library Management System developed using Python.
The program allows users to view available books, borrow books, return books, and check for overdue books.

The system simulates how a real library operates by keeping track of book availability, due dates, and fines.

---

##  Features

* View all available books
* Borrow a book
* Return a book
* Check overdue books and fines
* Simple menu-based interface

---

## Technologies Used

* Python (Core Python)


---

##  How to Run the Program

*Open terminal and run the code 

```
python main.py
```

---

## How the System Works

1. The system displays a menu with options.
2. Users select an action (view, borrow, return, overdue).
3. The system processes the request and displays the result.
4. Borrowed books are assigned a due date of 7 days.
5. Late returns are charged a fine of 2 per day.

---

## sample Operations

* Viewing available books
* Borrowing a book with User ID
* Returning a book and calculating fine
* Checking overdue books
